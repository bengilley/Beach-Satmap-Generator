# Краткое руководство RU

## Установка

Установите Python, затем зависимости:

```bash
py -m pip install -r requirements.txt
```

## Обязательные файлы

Launcher требует:

```text
heightmap.asc
mask.png
satmap.png
layers.cfg
```

Пути к файлам по умолчанию пустые. Выберите файлы вручную во вкладке `1. Файлы`.

## Текстуры

Можно выбрать vanilla-текстуру DayZ из списка и/или вручную вписать custom-текстуры.

Пример:

```text
Существующий пляж / берег     : hp_beach
Исходный песок для расширения : hp_sand
Целевая земля внутри          : пусто или опциональная текстура
```

Используйте:

```text
Проверить текстуры layers.cfg
```

чтобы проверить, что имена текстур существуют в `layers.cfg`.

## Генерация

Запуск выполняется только во вкладке:

```text
4. Запуск
```

Результаты создаются в:

```text
outputs/output_V...
```
